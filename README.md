# auth_poc
Access Management POC

## Getting Started

```shell
npm install
```

## Running

```shell
node index.js
```

Or even better, with [nodemon](https://github.com/remy/nodemon)
```shell
npm install -g nodemon
nodemon index.js
```

#### Ports used:

microservice 1447

authservice  1448

## Usage Example (using http)

```shell
http localhost:1447/hello bp-wtf:aaa
```
